# BTPN JP - Final Project


How to run:

Front end:
use command prompt on employee-app folder, and type
- npm install
- npm start
- open localhost:9000 on browser

Back end:
use command prompt on EmployeeApp folder, and type
- mvn spring-boot:run

I'm using Chrome browser to try my project.
